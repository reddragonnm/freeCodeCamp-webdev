Hello World!

Hello World in Python is a really easy one liner. In this example print is a function (which is taught later) 

print("Hello World!")
Functions

Functions are a way to group code together. They are used to make code more readable and easier to understand. Functions are also used to make code more reusable.

def call_print():
    print("Hello World!")

call_print() # OUTPUT: Hello World!

# function with parameters
def add_numbers(num1, num2):
    print(num1 + num2)

add_numbers(1, 2) # OUTPUT: 3

# return value from function
def return_some_number():
    return 5

print(return_some_number()) # OUTPUT: 5
